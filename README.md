#**Evaluation task**

##**Commands**

###**Start server:**

    ```
    npm run express
    ```

will run assembly to 'public' folder then

will run express server at http://localhost:3000

###**Start dev-server:**

    ```
    npm run dev-server
    ```

will run webpack dev-server

###**Build to 'public' folder:**

    ```
    npm run build
    ```
